## User Stories

### Analyst

1. As an analyst, I want to see the sentiment label for each news article in the list view so that I can quickly identify negative news.

2. As an analyst, I want to view and respond to colleagues’ comments on a news article so that we can discuss possible responses to PR crises.

3. As an analyst, I want the system to store my previous analysis results so that I can review historical sentiment trends.

4. As an analyst, I want to adjust the sentiment score if the AI classification is inaccurate so that the analysis results can be improved.

### PR Manager

5. As a PR manager, I want to view a dashboard showing the overall sentiment distribution of a brand so that I can evaluate the current brand health.

6. As a PR manager, I want to compare sentiment scores between our brand and competitors so that I can understand our relative reputation in the market.

7. As a PR manager, I want to bookmark important news articles so that I can easily access them later.

### User

8. As a user, I want to search for news articles using keywords so that I can track sentiment related to specific products or topics.

9. As a user, I want to create an account and log in so that our internal sentiment analysis data remains secure.

### Administrator

10. As an administrator, I want to manage team member permissions so that only authorised PR staff can publish comments.
